CI Setup & Automations

Prevent security anomalies in your pull requests before merging into staging branches.

Block vulnerable Soroban PRs

Include Soroban Guardrail scan in your GitHub Actions workflow. Returns status code 1 on new critical or high issues to assert high-integrity merges.

.github/workflows/guardrail.yml 
name: Soroban Guardrail
on: [pull_request]
jobs:
  scan:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - run: npx soroban-guardrail scan ./contracts --format github
